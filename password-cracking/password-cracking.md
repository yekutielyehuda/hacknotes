# Password Cracking

## John

A simple way to crack password hashes with a dictionary attack:

```text
john --wordlist=/usr/share/wordlists/rockyou.txt passwords.txt
```

### Converting Formats for John

```text
find / -name '*2john' 2>/dev/null
```
