## 5985 - WinRM

### Evil-WinRM

Connect to the target:

```bash
evil-winrm -i <IP> -u 'username' -p 'password'
```

Authenticate to the target:

```bash
crackmapexec winrm 10.10.10.161 -u 'svc-alfresco' -p 's3rvice'
```
