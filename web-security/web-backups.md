# Web Backups

## Web Backups

Sometimes people forget or they intentionally leave backup files on the web server. Here is a list of common web backups name conventions:

```text
file.ext~
#file.ext#
~file.ext
file.ext.bak
file.ext.tmp
file.ext.old
file.bak
file.tmp
file.old
file.bak~
file.tmp~
file.old~
file.ini.php
file.ini.conf
file.inc.php
file.inc.conf
```
